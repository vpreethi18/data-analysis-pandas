# Data Analysis using Pandas

## Objective
To analyze a CSV dataset, clean the data, and generate summary statistics using Python Pandas.

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## Dataset
Student performance dataset containing marks and department details.

## Steps Performed
- Loaded CSV file using Pandas
- Handled missing values and duplicates
- Filtered data based on conditions
- Grouped data to calculate summary statistics
- Derived meaningful insights

## How to Run
1. Clone the repository
2. Open `data_analysis.ipynb` in Jupyter Notebook
3. Run cells sequentially

## Output
The notebook displays cleaned data, grouped statistics, and insights.
